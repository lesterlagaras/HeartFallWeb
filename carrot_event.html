🥕 Carrot Event Announcement 🥕

Hello HeartFall players!

Get ready because a new Carrot Event is coming to the server! 🌟

🕒 Event Duration

5 minutes

💰 Price per Carrot

₱75 each

📌 How It Works

  • When the event starts, a server-wide broadcast will appear:
    🥕 CARROT EVENT STARTED! Sell your carrots for ₱75 each for 5 minutes! Use /sellcarrot to cash in!
  • Your goal is to collect and sell carrots within 5 minutes to earn as much money as possible.
  • Use /sellcarrot to convert your carrots into server currency.
  • When the 5 minutes are up, the server will broadcast:
    🥕 Carrot Event has ended. Back to normal selling prices!
  • New players joining while the event is active will see an action bar countdown.

⚡ Event Cycle

  • The Carrot Event will automatically activate every 1 hour.
  • Ops can also start it manually using the command /carrotevent start.

🎉 Tips

  • Stock up on carrots before the event starts to maximize your earnings!
  • Watch the action bar timer so you don’t miss the chance to sell!
  • Good luck, farmers! 🌱 Let’s see who can earn the most during the Carrot Event! 🥕💸
